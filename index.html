Introdução

JavaScript é uma linguagem de script orientada a objetos e multiplataforma.

É uma linguagem pequena e leve. Dentro de um ambiente hospedeiro, o JavaScript pode ser conectado aos objetos deste ambiente para prover controle programático sobre eles.

function cumprimentar(seuNome) { alert("Olá " + seuNome); }

Contém uma biblioteca padrão de objetos, como:

  • Array
  • Date
  • Math
O que você deveria saber

Este guia assume que você tem o seguinte conhecimento básico:

  • Compreensão geral de programação
  • Conhecimento básico de HTML
console.log('Olá Mundo!');
JavaScript e Java

JavaScript e Java são similares em alguns aspectos mas fundamentalmente diferentes em outros.

A linguagem JavaScript se assemelha ao Java mas não possui tipagem estática como Java.

let x = 42;
Olá mundo

Para começar a escrever JavaScript, abra seu editor favorito e escreva seu primeiro código "Olá mundo":

function cumprimentar(seuNome) { alert("Olá " + seuNome); }

Selecione o código e execute-o no seu navegador!

Variáveis

Você usa variáveis como nomes simbólicos para valores em sua aplicação.

Variáveis em JavaScript são declaradas de três formas:

  • Usando var
  • Usando let
  • Usando const
const PI = 3.14;
